
## Expense Tracker

A simple Expense Tracker web application to record, categorize, and visualize daily expenses with an easy-to-use interface.

## Features

Add expenses with description, amount, date, and category
View total expenses (last 30 days)
See daily average spending
Category-wise expense visualization using a pie chart

## Tech Stack

React.js
Vite
JavaScript
React-Bootstrap
Chart library (for pie chart visualization)

## Getting Started

Clone the repository
  git clone https://github.com/sharon1008/Expense-calculator.git
  cd Expense-calculator
Install dependencies
  npm install
Run the project
  npm run dev
App runs at:
  http://localhost:5173
